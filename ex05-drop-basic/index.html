jQuery UI - droppable avec options

Voici un exemple d'élément "droppable" qui présente quelques options pour rendre l'interaction plus intéressante. Le carré rouge ne sera pas accepté par l'élément cible et va retourner à sa place.

Drag me!

Le carré bleu, au contraire, sera accepté s'il sera trainé complètement à l'intérieur de l'élément cible.

Drag me!
Drop inside this area!

Voici le code utilisé pour cet exemple :

<script>
$("#redBox").draggable({
	revert: 'invalid'
});
$("#bluBox").draggable({
	revert: 'invalid'
});
$("#blackBox").droppable({
	accept: '#bluBox',
	tolerance: 'fit'
});
</script>
