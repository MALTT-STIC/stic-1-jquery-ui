jQuery UI - élément "draggable"

Voici un simple interaction qui permet de traîner un élément avec la souris. Cliquer sur le carré rouge et trainé-le en laissant appuyé le bouton de la souris. Comme vous pouvez le voir, le carré rouge peut être trainé n'importe où dans la page.

Drag me!

Voici le code utilisé pour cet exemple :

<script>
$("#redBox").draggable();
</script>
