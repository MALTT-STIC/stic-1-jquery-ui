jQuery UI - drag avec code

Dans cette exemple, les événements start et stop de la fonction draggable() sont associés à une fonction qui change le texte dans le paragraphe feedback.

Drag me!

Voici le code :

<script>
   $("#redBox").draggable({
 start: function() {
 $("#feedback").text("J'ai commencé le drag");
 },
 stop: function() {
 $("#feedback").text("J'ai terminé le drag");
 }
 });
 </script>
