jQuery UI - datepicker avec format de la date

Voici un deuxième exemple : un champ de texte qui propose automatiquement un calendrier superposé pour choisir une date, et cette date a un format différent à celui utilisé par défaut. Cliquez à l'intérieur du champ de texte pour faire apparaître le widget.

Date

Contrairement à l'exemple 1, vous pouvez noter que la date a un format de type jour-mois-année, comparé au format mois/jour/année de l'exemple précédent. Pour obtenir ce résultat, il suffit d'ajouter des options à l'intérieur de la fonction JavaScript : 

<script>
$("#mon-champ-date").datepicker({
	dateFormat: "dd-mm-yy"	
});
</script>
