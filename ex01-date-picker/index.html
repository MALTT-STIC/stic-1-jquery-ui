jQuery UI - datepicker simple

Voici un premier exemple : un champ de texte qui propose automatiquement un calendrier superposé pour choisir une date. Cliquez à l'intérieur du champ de texte pour faire apparaître le widget.

Date

Pour obtenir ce type de résultat il suffit de donner un ID au champ de texte, dans notre cas "mon-champ-date", et ensuite ajouter le code JavaScript suivant. 

<script>
$("#mon-champ-date").datepicker();
</script>
